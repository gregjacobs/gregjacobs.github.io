Hello, my name is Gregory Jacobs. I am a software engineer with years of experience in web application development.

I mainly specialize in developing rich, front end applications using modern front-end libraries such as React along with the architecture/patterns to design efficient and maintainable front-end systems.

In my career I've been both a full stack engineer (ColdFusion / Java / PHP / ASP / MSSQL on the backend), and fully-dedicated frontend engineer. In addition to working extensively with React and Angular on the front-end, I've also worked with many past front-end libraries and frameworks such as AngularJS, ExtJS, jQuery, and YUI.

I have also had a large focus on and have deep knowledge of: JavaScript build tooling, CI pipelines, test tools, effective testing patterns, codemods, monorepos + scaling for large projects, and many other areas of Web, Node.js, and other development. In addition, I enjoy contributing to Java, Go, and other projects when the opportunities arise.
